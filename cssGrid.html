Auto-fit and minmax

Auto-fit will add and remove columns/rows as more space is available. Alone, this will create extra space though. Use minmax() to manage the minimum and maximum values for multiple items in a row/column. Note that the resulting implict creation may not fit all contents when you resize smaller.

Implicit rows

Use grid-auto-rows instead of grid-template-rows to make explicitly define the size of the rows. In this way, you can scale the screen size down but keep the same sized row.

Awesome Image Grid

Use grid-auto-flow property 'dense' to fill in the black spots in a container. This is an example of Source Order Independence.

Named Lines

Use properties in the container, and reference them in items for modular layouts. Items that have Dash-separated suffixes (ie. start, end) will allow you to just specify what was before the dash in the respective class. You can use grid-area instead of grid-column/row if you have properly boxed in the content.
